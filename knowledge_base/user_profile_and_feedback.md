# User Profile and Feedback System

## Profile Overview
Each student has a verified **college ID account**.

Profile contains:
- Name
- College ID
- Branch / Year
- Interests
- Profile Picture

## Notifications
Students receive notifications for:
- New Lost & Found posts
- Upcoming events
- Comments or replies on their posts
- Important college announcements

Notification preferences can be managed under **Profile → Settings**.

## Reporting Bugs or Issues
- Go to **Profile → Feedback**.
- Fill out the feedback form describing your issue or suggestion.
- The issue is sent directly to **admins** for review.

## Moderation
Admin team reviews:
- Reported posts
- Lost & Found spam
- Feedback forms

Admins ensure content quality and help maintain a positive app experience.
